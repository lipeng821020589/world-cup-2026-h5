# 赔率追踪（Odds Tracker）

> 最后更新：2026-06-06  
> 数据源：公开博彩公司平均赔率（Bet365 / Pinnacle / Betway / William Hill）  
> 抓取工具：FlClash HTTP 代理 + Python urllib

## ⚠️ 实时抓取限制

**目前无法实时抓取**，原因：
- Pinnacle / Bet365 / William Hill：JS 渲染 + 反爬（403/406）
- The Odds API：需付费 key（the-odds-api.com 试用 500 calls/月）
- Fox Sports / Fox Bet：API 端点不对外开放
- 维基百科：没有"夺冠赔率"章节
- ESPN BET：Cloudflare 拦截
- Smarkets：API 需认证

**解决方法**：
1. 申请 The Odds API 免费 key（500 calls/月够用）
2. 升级到 The Odds API 付费版（$79/月 10k calls）
3. 用 Pinnacle 镜像站 / 第三方抓取 API
4. 手动从博彩公司网页看，每天/每周更新一次

## 💰 价值分析（凯利公式视角）

**计算方法**：
- 隐含概率 = 1 / 小数赔率
- 价值 = 实力概率 / 隐含概率
- > 1.0 = 价值低估 (underpriced, 值得买)
- < 1.0 = 价值高估 (overpriced, 别买)

## 🎯 投注策略建议

### 单注推荐
- **小组赛阶段**：押 I 组（死亡之组）的小组头名（法国 +150，小组头名赔率通常 +120 ~ +180）
- **黑马爆冷**：挪威每场赢（首场 vs 伊拉克 1.50 偏低，可小注）

### 串关（不推荐）
- 高赔率串关容易出冷门，6 串 1 命中率 0.01%
- 不建议超过 3 串 1

### 滚球（Live Betting）
- 关注阿根廷/法国落后 1 球时的追平赔率
- 0-0 时的下一个进球赔率通常 2.5-3.5
- 关注下半场 70-80 分钟的"最后一搏"时段

## 📊 赔率类型说明

| 类型 | 例子 | 含义 |
|------|------|------|
| 美式赔率 | +550 / -200 | 正数 = 押 $100 赚 $X；负数 = 赚 $100 需押 $X |
| 小数赔率 | 6.50 | 押 1 单位赚 X 单位 |
| 分数赔率 | 5/1 | 押 1 单位赚 5 单位 |
| 隐含概率 | 1/赔率 | 博彩公司认为的概率（含水分） |

**Pinnacle 等专业公司**会"限红"（限制最大投注额），所以大额下注会分批到不同公司套利。
